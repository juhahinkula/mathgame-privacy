# Kids Math Adventure — Support

If you need help with Math Adventure (publishing, store review, bugs, or general support), use the contact options below.

Contact
- Email: juha.hinkula@tutamail.com

What to include when contacting support
- App version (e.g. 1.0.8)
- Device model and OS version (e.g. iPhone 12, iOS 17.2)
- Steps to reproduce the issue
- Expected vs actual result
- Screenshots or screen recordings (attach)

For App Store reviewers
- iOS workspace: ios/Runner.xcworkspace
- Android resources: android/app/src/main/res
- Test account credentials (if required): include in message

Quick troubleshooting (try before contacting support)
1. Restart the app.
2. Update to the latest version from the store.
3. Reboot the device.
4. Reinstall the app.

Notes on data and privacy:
Math Adventure stores only local preferences (player name, age, settings). No analytics or personal data is collected unless you explicitly send logs or screenshots.

Thank you — including the details above helps us resolve issues faster.
